# TheNoteApp

A simple small Note-taking-web-app .

## Features
- Create Note.
- Edit Note.
- Delete Note.
- Regular Expression for Register and SignIn.
- Save the token in localstorage.
- Responsive.
## Technologies
- HTML
- CSS
- JavaScript
- Bootstrap 
- Font Awesome 
- React Js
- Axios
- Jwt-Decode
- Note Api
